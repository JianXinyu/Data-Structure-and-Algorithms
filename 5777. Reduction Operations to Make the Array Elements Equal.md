# Solution 1
观察发现，答案只和比最小值大的元素数目有关。
比如\[1,2,3,4,5\]
1. 只有一个5，操作一次\[1,2,3,4,**4**\]
2. 有两个4，操作两次\[1,2,3,**3,3**\]
3. 有三个3，操作三次\[1,2,**2,2,2**\]
4. 有四个2，操作四次\[1,**1,1,1,1**\]
总共操作10次

- 统计每个元素的数目
- 从最大值开始操作
- 增加下一个最大元素数目
```cpp
class Solution {
public:
    int reductionOperations(vector<int>& nums) {
        map<int, int> m;
        for(int i = 0; i < nums.size(); ++i){
            m[nums[i]]++;
        }
        if(m.size() == 1)
            return 0;
        int ans = 0;
        map<int, int>:: iterator last;
        for(auto itr = m.end(); itr != m.begin(); --itr){
            if(itr != m.end())
                itr->second += last->second;
            ans += itr->second;
            last = itr;
        }
        return ans;
    }
};
```
T: O(nlogn)
S: O(n)

# 官方题解
```cpp
class Solution {
public:
    int reductionOperations(vector<int>& nums) {
        sort(nums.begin(), nums.end());
        int n = nums.size();
        int res = 0;   // 总操作次数
        int cnt = 0;   // 每个元素操作次数
        for (int i = 1; i < n; ++i) {
            if (nums[i] != nums[i-1]){
                ++cnt;
            }
            res += cnt;
        }
        return res;
    }
};
```
T: O(nlogn) 排序
S: O(logn) 排序的栈空间开销
